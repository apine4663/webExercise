What is HTTP? -A protocol ran when a machine tries to connect to another machine
What is a URL? -Uniform Resource Locator is an address assigned to a resource on the web
What is DNS? -The Domain Name System is basically the phonebook of the internet and keeps a record of all web names
What is a query string? -A search term added to the end of the url to specify what you're looking for
What are two HTTP verbs and how are they different? -GET- a retrieval of data from a server -POST- a change or addition to data in a server
What is an HTTP request? -A request from user to a server using the HTTP protocol
What is an HTTP response? -A response from a server to a user using HTTP protocol
What is an HTTP header? Give a couple examples of request and response headers you have seen. -Headers give information on request or the response. -Request headers: Host, User-Agent, Accept, Cookie, Cache-Control -Response headers: Content-Type, Last-Modified, Set-Cookie, Cache-Control
What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser? 1. The protocol is determined 2. The hostname IP is attained from the DNS 3. A request is made on the server. 4. A response is given to the user 5. Page is loaded with info from the response. 6. Requests and Responses are made for subsequent resources on page.
